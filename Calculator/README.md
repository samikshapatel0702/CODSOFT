# Simple Calculator

## Description

This is a simple calculator developed using Python as part of the CodSoft Python Programming Internship. The calculator performs basic arithmetic operations and provides a user-friendly menu-driven interface.

## Features

* Addition
* Subtraction
* Multiplication
* Division
* Power Calculation
* Modulus Operation
* Error Handling for Division by Zero
* Continuous Calculations Until Exit

## Technologies Used

* Python

## How to Run

1. Open the project in VS Code.
2. Run the `calculator.py` file.
3. Select an operation from the menu.
4. Enter the required numbers.
5. View the result.

* Addition

* Multiplication
