# Rock Paper Scissors Game

## Description

This is a Python-based Rock Paper Scissors game developed as part of the CodSoft Python Programming Internship. The game allows a user to play against the computer, which makes random choices each round.

## Features

* User vs Computer gameplay
* Random computer choice generation
* Score tracking system
* Multiple rounds support
* Winner determination based on game rules
* User-friendly interface

## Game Rules

* Rock beats Scissors
* Scissors beats Paper
* Paper beats Rock
* If both choose the same option, the round is a tie

## Technologies Used

* Python
* Random Module

## How to Run

1. Open the project in VS Code.
2. Run the `rock_paper_scissors.py` file.
3. Enter Rock, Paper, or Scissors when prompted.
4. View the result and continue playing if desired.
