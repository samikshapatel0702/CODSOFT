# CODSOFT Python Programming Internship

## Overview

This repository contains the projects completed during the CodSoft Python Programming Internship. The projects focus on strengthening Python programming skills through practical applications and problem-solving.

## Projects

### Task 1: Calculator

A menu-driven calculator that performs:

* Addition
* Subtraction
* Multiplication
* Division
* Power Calculation
* Modulus Operation

### Task 2: Password Generator

A password generator that:

* Generates secure random passwords
* Allows custom password lengths
* Supports letters, numbers, and special characters

### Task 3: Rock Paper Scissors Game

An interactive game where:

* Users play against the computer
* Scores are tracked
* Multiple rounds can be played
* Winners are determined according to game rules
